Ключевые возможности

Чанковая загрузка

Файлы до 100 ГБ с возобновлением и прогрессом. Группировка в пакеты, публичные ссылки с лимитами.

Локальный чат

Общий чат и личные сообщения, упоминания, эмодзи, Markdown, уведомления в реальном времени.

Голосовой чат

Комнаты, Push‑to‑Talk, шумоподавление, регулировка громкости каждого участника.

3D просмотрщик

STL, OBJ, GLTF, PLY, FBX – вращайте, приближайте, включайте каркасный режим.

Кастомизация темы

Цвета, радиус скругления, скорость анимации, шрифты – создайте свой уникальный стиль.

Безопасность

2FA, роли (админ/пользователь), блокировка аккаунтов, CSRF‑защита.

Быстрый старт

1
Клонируйте репозиторий 
git clone https://github.com/fedorramezov-del/Space_Share/tree/main.git
cd space-share
2
Установите зависимости 
pip install -r requirements.txt
3
Подготовьте SSL-сертификаты (для HTTPS)

Если у вас уже есть файлы cert.pem и key.pem, поместите их в корень проекта. Если нет — сгенерируйте самоподписанные:

openssl req -x509 -newkey rsa:4096 -nodes -out cert.pem -keyout key.pem -days 365
4
Запустите сервер 
python wsgi.py

Откроется графическое окно. Нажмите «Запустить сервер» и перейдите по адресу https://localhost:5000.

5
Настройте аккаунт

Зарегистрируйте первого пользователя — он получит права администратора. Наслаждайтесь работой с файлами, чатом и 3D.
